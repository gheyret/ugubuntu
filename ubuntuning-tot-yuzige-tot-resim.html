ھەممىمىزگە مەلۇم Ubuntu نى ئادەتتە تۆت ئۈستەل يۈزى قىلىپ ئىشلىتىمىز (تۆتتىن جىق ياكى ئاز قىلسىمۇ بولىدۇ ئەمدى). بىراق ئۈستەل يۈزىگە رەسىم تاللىغاندا پەقەت بىرنىلا تاللىغىلى بولىدۇ، يەنى ھەممە يۈزدىكى رەسىم ئوخشاش. مەن تۆۋەندە CompizConfig Settings Manager دىن پايدىلىنىپ Ubuntu نىڭ قانچە ئۈستەل يۈزى بولسا ئۈستەل يۈزىگە ئوخشىمايدىغان شۇنچە خىل رەسىم ئىشلىتىشنى تونۇشتۇرۇپ ئۆتەي.
ئالدى بىلەن System–>Preferences–>CompizConfig Settings Manager->Wallpaper
gconf-editor غا كىرىمىز. ( CompizConfig ھەممىمىزدە باردۇر). ئاندىن قانچە ئۈستەل يۈزى بولسا ئۈستەل يۈزىگە ئوخشىمايدىغان شۇنچە خىل رەسىمنى تاللايمىز.

تېرمىنالغا  gconf-editor نى كىرگۈزىمىز ياكى مائۇسنى ئېكراننىڭ سول يۇقىرى تەرىپىدىكى Application نىڭ ئۈستىگە ئەكېلىپ ئوڭ كۇنۇپكىسىنى باسىمىز. ئاندىن چىققان تىزىملىكتىن Edit Menu نى تاللاپ ئېچىلغان كۆزنەكتىن System Tools نى تېپىپ ئوڭ تەرەپتىن Configuration Editor نى تاللايمىز دە Application ——> Seytem Tools__>Configuration Editor غا كىرىمىز.
سول تەرەپتىن تەرىپ بويىچە /apps/nautilus/prefrence/ مۇندەرىجىسىگە كىرىپ، ئوڭ تەرەپتىكى show_desktop نى تاقىۋېتىمىز (ئالدىدىكى تاللاش بەلگىسىنى چىقىرىۋېتىمىز).

قانداق؟! دېگىنىمدەك سىز تاللىغان رەسىملەر ھەر بىر ئۈستەل يۈزىدە ئايرىم كۆرۈندىمۇ؟
