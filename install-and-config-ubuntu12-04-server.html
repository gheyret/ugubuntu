Ubuntu 12.04 مۇلازىمېتىرىنى قاچىلاش، ۋە دەسلەپكى تەڭشەش
ئېلان قىلىنغىلى245 كۈن بولدى / لىنۇكس مۇلازىمېتىرى / 10 پارچە ئىنكاس چۈشتى / كۆرۈلگىنى 2,365+

(Ubuntu 12.04 Server (LTS بولسا Ubuntu نىڭ مۇلازىمېتىر نۇسخىسى. قېرىنداشلىرىمىز ئارىسىدا Linux مۇلازىمېتىرىغا ھاجىتى چۈشىدىغانلارنىڭ كۆپىيىشىگە شۇنداقلا Ubuntu ئۈستەل يۈزى نۇسخىسىنى تونۇشتۇرىدىغان بېكەت بىلوگلارنىڭ كۆپىيىشىگە ئەگىشىپ، ئۇيغۇرچە Linux مۇلازىمېتىرىنى تونۇشتۇرىدىغان بېكەت بىلوگلارنىڭ ئاز بۆلىشىدەك بوشلۇقنى تولدۇرۇشقا پايدىسى بولۇپ قالار دەپ، بىلوگقا ئاساسلىق Ubuntu مۇلازىمېتىرى ھەققىدىكى يازمىلارنى يېزىپ قويغىلى «لىنۇكس مۇلازىمېتىرى» دېگەن كاتېگورىيىنى قوشتۇم.

  • قاچىلاش تەييارلىقىنى چۈشەندۈرۈش

مەن قاچىلىماقچى بولغان Ubuntu مۇلازىمېتىر ئىسمى (Host主机) بولسا: server.example.com
IP ئادرېسى: 10.42.0.42
تور ئۆتكىلى (Host主机) بولسا : 10.42.0.1
سىناق قىلغان مۇھىت: VirtaulBOX مەۋھۇم كومپيۇتېرى
ئەلۋەتتە بولارنىڭ ھەممىسى قاچىلاش مۇھىتىڭلارغا قاراپ ئۆزگىرىدۇ.

قاچىلاش جەريانىدا ئۈستى ئاستىنى تالاش ئۈچۈن ئۈستى ئاستى يۆلىنىش كۇنۇپكىسىنى، ئوڭ سول، ۋە بەزى جەزملەشتۈرۈش چېكىنىش (Yes/NO/Go back) دېگەنلەرنى تاللاش ئۈچۈن Tab كۇنۇپكىسىنى، تاللىغان تاللاشنى جەزملەشتۈرۈش ئۈچۈن ئېنتىر باسىمىز.

  • قاچىلاش

1. Ubuntu 12.04 Server نى چۈشۈرۈپ، كومپيۇتېرنى ISO ئويۇلغان ئوپتىك دىسكىدىن ياكى بارماق دىسكىدىن قوزغىتىمىز. ھەم قاچىلاش جەريانىدا ئىشلىتىدىغان تىلنى تاللايمىز. مەن بۇ يەردە ئىنگلىزچىنى تاللىدىم.

2. بۇ يەردىن Ubuntu نى قاچىلاش يەنى  Install Ubuntu Server دېگەننى تاللىدىم.

3. بۇ يەردىن سىستېما تىلىنى تاللايمىز، Ubuntu نىڭ مۇلازىمېتىر نۇسخىسىدا سۈكۈت ھالەتتە ئۈستەل يۈزى بولمىغاچقا (بۇيرۇق ئېكران) بۇ يەردىن ئىنگلىزچىنى تاللاپ قاچىلىدىم. ئەلۋەتتە بۇ يەردىن باشقا تىلنى تاللاپ قاچىلىساق بولىدۇ. پەقەت بىر قىسىم بۇيرۇق دېتاللارنىڭ ئۇچۇرىلا شۇ تىلدا كۆرۈنىدۇ.

4. تۇرۇۋاتقان رايوننى تاللايمىز. تۇنجى كۆرۈنگەن تىزىملىكتە جۇڭگو بولمىغاچقا باشقىلار Other دېگەننى تاللىدىم.

5. كېيىنكى كۆرۈنگەن تىزىملىكتىن باشتا ئاسىيا Asia ئاندىن جۇڭگو China نى تاللىدىم.

6. ئىشلىتىدىغان كۇنۇپكا تاختىسىنى تاللايمىز. بىز ئامېرىكا كۇنۇپكا تاختىسىنى United States en_US.UTF-8 تاللىساق بولىدۇ.

7. بۇ كۇنۇپكا تاختىسىنى سىنايدىغان يەر بولۇپ، Yes نى تاللاپ، كۆرۈنگەن بىر قانچە ھەرپنى بېسىپ كۇنۇپكا تاختىمىزنى بەلگىلىسەكمۇ، ياكى NO نى بېسىپ، ئۆزىمىز تاللىساقمۇ بولىدۇ. مەن NO نى تاللىدىم.

ئاندىن كۆرۈنگەن ئىككى كۆزنەككە ئامېرىكا ئىنگلىزچىسىنى (English (US نى تاللىدىم.
