ئېلان قىلىنغىلى210 كۈن بولدى / يۇمشاق دېتال تەۋسىيىسى / بىر پارچە ئىنكاس چۈشتى / كۆرۈلگىنى 631+

ClassicMenu Indicator بولسا App-Indicator قورال ئىستونىدىكى Gnome تارتىلما تىزىملىكى. يەنى ئۇبۇنتۇنىڭ بۇرۇنقى تارتىلما تىزىملىكى. Unity ئارايۈزىدىن بىزار بولغانلار ئۈچۈن ناھايىتى ياخشى ياردەمچى.

تۈر ئورگان بېتى: http://www.florian-diesch.de/software/classicmenu-indicator

قاچىلاش بۇيرۇقى:

sudo add-apt-repository ppa:diesch/testing
sudo apt-get update
sudo apt-get install classicmenu-indicator

قاچىلاش مۇۋەپپەقىيەتلىك بولغاندىن كىيىن classicmenu-indicator قوزغىتىپ ئىشلەتسەڭلار بولىدۇ. ئۈنۈمى رەسىمدىكىدەك. سىستېما بىلەن تەڭ قوزغىلىدىغان قىلىش ئۈچۈن قوزغىلىش تىزىملىكىگە ئايرىم قوشۇۋالساڭلارمۇ بولىدۇ.
