ئوبۇنتۇدا كوميىوتىرنىڭ قاتتىق دىتالنىڭ ئۇچۇرىنى كۈرۈش ئۇسۇلى

بۇنىڭ ئۇسۇلى ناھايىتى ئادى بۇلۇپ تۈۋەندە مەن ئۈچ خىل ئۇسۇلىنى تۇنۇشتۇرۇپ ئۈتمەن.

1.lshw     بىلەن         lshw-GTK نى قاچىلاش ئارقىلىق ئېرىشىش

lshw بولسا ئاساسى تاختا،سىپىيو ،بىئوسلارنىڭ ئۇچۇرنى چىقىرىپ بىرىدۇ.

قاچىلاش ئۇسۇلى :

sudo apt-get install lshw lshw-gtk  

بۇ دىتال قاچىلىنىپ كۈرنىشى مۇنداق چىقىدۇ.


2.  hardinfo  نى قاچىلاش ئارقىلىق ئىرشىش 

قاچىلاش ئۇسۇلى :

sudo apt-get install hardinfo 



3 .sysinfo  قاچىلاش ئارقىلىق ئىرشىش 

قاچىلاش ئۇسۇلى  :

sudo apt-get install sysinfo  



قالغنىنى قىزقىدىغانلار قاچىلاپ كۈرۈپ باقسا بۇلىدۇ.بۇلارنىڭ ھەممىسىنىڭ ئالاھىدىلىكى ئوخشاش بۇلۇپ پەقەت ئۇچۇرلانى چىقىرىپ كۈرسىتىپ بىرىش جەھەتتكى قاتتىق دىتاللارنى تۇنۇش ئىقتىدارى پەرقىلىنىدۇ خالاس.
