Linux Mint 11 تارقىتىلدى
ئېلان قىلىنغىلى566 كۈن بولدى / لىنۇكس يېڭىلىقلىرى / 3 پارچە ئىنكاس چۈشتى / كۆرۈلگىنى 1,020+

بىلوگدىكى يازمىلارنى تولۇق كۆرۈپ كىلىۋاتقانلار Linux Mint نىڭ قانداق سىستېما ئىكەنلىكىنى بىلسە كېرەك. چۈنكى مەن بۇرۇنقى بىر نەچچە يازمىلىرىمدا Linux Mint نى تىل ئېلىپ ئۆتكەن. Mint لىنۇكس بولسا ئۇبۇنتۇ ئاساسىغا قورۇلغان بولۇپ، 2006-يىلى Clement Lefebvre تەرىپىدىن ئېچىلغان. Mint پۈتۈنلەي ھەقسىز بولۇپ، ئۇنىڭ بارلىق چىقىمى ئىئانە ۋە ئېلان كىرىمىدىن كەلگەن. شۇڭا مۇقىم يېڭىلىنىش ۋاقتى يوق. مۇشۇ ئايدا ئۇنىڭ ئەڭ يېڭى نەشىرى Linux Mint 11 ئېلان قىلىندى. ئۇنى ئېشلىتىش ئۇبۇنتۇغا ئىشلىتىشكە ئوخشاش بولۇپ، ئۇبۇنتۇ 11.04 نىڭ Unity ئارا يۈزىگە كۆنەلمىگەنلەر ئىشلىتىپ باقساڭلار بولىدۇ.

ئورگان بېتى: http://linuxmint.com

قايسى كۈنى Linux Mint 11 نىڭ iso ھۆججىتىنى بارماق دىسكامغا ئۇيۇم، شۇنىڭدىن قوزغۇتۇپ كۆرۈپ باقتىم، كۆرۈنىشىنى ھەقىقەتەن پىچەت لاھىيلەپتۇ. كۆرۈپ بېقىڭلار.
