مەن تۆۋەندە ئۇبۇنتۇدا ئۇيغۇرچە كىرگۈزۈشنىڭ ئەڭ ئاسان ئۇسۇلى يەنى سىستېمىغا سەپلەگلىك بولغان IBus ئارقىلىق ئۇيغۇرچە كىرگۈزۈشنى تونۇشتۇرۇپ ئۆتىمەن. ‏IBus سىستېمىغا سەپلەگلىك بولغان يۇمتال بولۇپ ئەگەر ئۆچۈرۋەتكەتلەر بولسا تۆۋەندىكى بۇيرۇق ئارقىلىق قاچىلىساڭلار بولىدۇ. ئەگەر سىستېمىدا سەپلەگلىك بولسا قاچىلاش ھاجەتسىز. ئەڭ ياخشىسى System>>Prefence نى ئېچىپ قاراپ بېقىڭلار.

قاچىلاش بۇيرۇقى:

www.uyghur-archive.com/ugubuntuputer:~$sudo apt-get install ibus

ئاندىن يۇمتال قاچىلاش ئورنى بولغان ‏System>>Administration>>Synaptic Package Manager نى ئېچىپ ئۇنىڭدىن IBus m17n قىستۇرما دېتالىنى چۈشۈرىمىز. ئادەتتە ubuntu 10.04 ئىشلىتىۋاتقانلار بۇ باسقۇچنى قىلمىسىمۇ قىستۇرما دېتال سىستېما بىلەن بىرگە IBus قا قاچىلاقلىق. بۇ قىستۇرما دېتالدا دۇنيادىكى ھەممە تىل دېگۈدەك بار. بىز كۆزنەك ئۈستىدىكى ئىزدەش رايونىغا IBus m17n دەپ كىرگۈزۈپ رەسىمدە كۆرسىتىلگەندەك ئۇنى توردىن چۈشۈرۈپ قاچىلايمىز.

Ibus نى قايتىدىن قوزغىتىپ Input Method دېگەن ئورۇندىن، ئۇيغۇرچىنى تاللاپ Add نى بېسىپ قوشساقلا بولدى.

ئەمدى IBus نى سىستېما بىلەن تەڭ قوزغىلىدىغان قىلىپ بەلگىلەيمىز. ئالدى بىلەن System>>Prefence>>Start Up Aplication نى ئىچىپ بىرىنچى قۇرغا ئىسمىنى ئىككىنچى قۇرغا IBus نى قوزغىتىش بۇيرۇقى /usr/bin/ibus-daemon -d نى كىرگۈزىمىز (ئادەتتە سىستېما قاچىلىغاندىلا بۇ يەر تەڭشەكلىك).

يۇقىرىدىكى مەشغۇلات ئوڭۇشلۇق بولغاندىن كىيىن ibus نى قايتا قوزغاتساق ياكى سىستېمىدىن چېكىنىپ ياكى قايتا قوزغىتىپ كىرسەكلا بولدى. تىل ئالماشتۇرۇش ئۈچۈن Ctrl + Space نى باسىمىز. سىستېمىدا ئۇيغۇرچە خەت توغرا كۆرۈنۈشى ئۈچۈن كومپيۇتېرغا چوقۇم ئۇيغۇرچە Unicod خەت شەكلى قاچىلانغان بولۇشى كېرەك.

بىز جۇڭگودا ياشايدىكەنمىز كومپيۇتېرىمىزدا خەنچە كىرگۈزگۈچ بولمىسا بولما. Ibus سەپلەنگەن پىنيىنچە كىرگۈزگۈچ ياخشى ئەمەس. نىڭ m17n قىستۇرما دېتالىدىكى خەنزۇچە كىرگۈزگۈچ پەقەت ياخشى بولمىغانلىقتىن مەن تۆۋەندە windows دىكى 搜狗 كىرگۈزگۈچى بىلەن ئانچە پەرقلەنمەيدىغان خەنزۇچە كىرگۈچنى IBus قا قاچىلاشنى تونۇشتۇرۇپ ئۆتمەكچى. ئالدى بىلەن System>>Administration>>Synaptic Package Manager نە ئېچىپ ئىزدەش رايونىغا IBus pinyin دەپ كىرگۈزىمىز، ئاندىن رەسىمدە كۆرسىتىلگەن قىستۇرما دېتاللارنى قاچىلايمىز.

ئاندىن IBus نى ئېچىپ بايا ئۇيغۇرچىنى قوشقاندەك Pinyin نىمۇ قوشىمىز.
