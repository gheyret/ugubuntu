ئۇيغۇرچە خەت نۇسخىلىرىنى قاچىلاپ بولغاندىن كىيىن، شۇ خەت نۇسخىلىرىنىڭ نورمال قاچىلانغان قاچىلانمىغانلىقىنى سىناپ بېقىڭ، مەسىلەن تېرمىنال، gedit, office قاتارلىقلارنىڭ خالىغان بىرسىنىڭ خەت نۇسخىسىنى تەڭشەيدىغان يىرىدە سىز قاچىلىغان ئۇيغۇرچە خەت نۇسخىلىرىنىڭ بار يوقلىقىغا قاراپ بېقىڭ.

ئەگەر ئۇيغۇرچە خەت نۇسخىلىرى نورمال قاچىلىنىپ، سىستېمىنى ئۇيغۇرچە قىلغاندا، كۆزنەكلەردىكى خەت نورمال كۆرۈنمىسە، سىز سىستېمىنىڭ خەت نۇسخىلىرىنى ئۇيغۇرچىغا ئالماشتۇرىشىڭىز كېرەك. ئۇبۇنتۇ 11.01 دە Advance Tools قورالىنى ئىشلىتىپ، خەت نۇسخىلىرىنى ئۇيغۇرچىغا تەڭشىۋالسىڭىز بولىدۇ.
قاچىلاش بۇيرۇقى:
Code:
sudo apt-get install gnome-tweak-tool

ئەگەر دېمەكچى بولغىنىڭىز، ئۇيغۇچە خەت نۇسخىلىرى قاچىلىنىپ، firefox دا، ئۇيغۇرچە تور بېكەتلەرنىڭ خېتى نورمال كۆرۈنمىسە، firefox نى بارلىق خەت نۇسخىلىرىنى كومپيۇتېردىن ئالدىغان قىلىپ تەڭشىۋالسىڭىز بولىدۇ. بۇ يازمامنى كۆرۈپ بېقىڭ. Firefox نىڭ يېڭى نەشرىدە ئۇيغۇچە خەتلەرنىڭ نورمال كۆرۈنمەسلىك مەسىلىسىنى ھەل قىلىش
