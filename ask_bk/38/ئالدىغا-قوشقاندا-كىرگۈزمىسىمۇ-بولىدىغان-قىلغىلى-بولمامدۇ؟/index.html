ئەلۋەتتە بولىدۇ. سوئالىڭىزدىن قارىغاندا sudo نىڭ نىمە ئىكەنلىكىنى ياخشى بىلىدىكەنسىز. شۇڭا جىق نەزىرىيە سۆزلىمەي.

سىز etc/sudoers/ ھۆججىتىگە نىسپەتەن تەھرىرلەش ئېلىپ بارسىڭىز بولىدۇ. يەنى تېرمىنالغا تۆۋەندىكى ئىككى قور بۇيرۇقنىڭ خالىغىننى كىرگۈزۈپ vi بىلەن شۇ ھۆججىتنى ئېچىڭ.
Code:
sudo vi /etc/sudoers
Code:
sudo visudo

ئاندىن ھۆججىتنىڭ خالىغان يىرىگە ISIM ALL = NOPASSWD: ALL دەپ قوشۇپ ساقلار قويسىڭىزلا بولىدۇ. تۆۋەندىكىسى مىنىڭ تەڭشىگىنىم.
