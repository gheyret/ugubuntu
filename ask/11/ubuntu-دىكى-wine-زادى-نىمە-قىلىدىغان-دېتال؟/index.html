Ubuntu دىكى Wine زادى نىمە قىلىدىغان دېتال؟

0 نومۇر
Ubuntu نى نەچچە كۈن ئاۋال قاچىلىغان، بىراق تازا كۆنەلمىدىم. Windows دىكى دېتاللارغا بەك كۆنۈپ قاپتىمەن. Windows دىكى بەزى دېتاللارنى Ubuntu دا ئىشلەتكىلى بولارمۇ دەپ توردىن ئىزدىسەم Wind دېگەن دېتالنى تەۋىسسىيە قىپتۇ، بىراق پەقەت چۈشۈنمىدىم، بۇ دېتالنىڭ زادى نىمە رلى بار؟ قانداق قاچىلاپ ئىشلىتىدۇ؟

Wine بولسا Linux مەشغۇلات سىستېمىسدا Windwos پروگراممىلىرىنى ئىشلىتىشتە ئىشلىتىلدىغان دېتال. بۇ دېتالنىڭ ئالاھىدىلىكى كومپيۇتېرىڭىزغا ئايرىم مەۋھۇم كوميۇتېر قۇرۇپ، ئۇنىڭغا Windows قاچىلاپ ئاندىن Windows دېتاللىرىنى ئىشلىتىشتەك ئاۋارىچىلىقلاردىن ساقلىنىپ قالىسىز، قاچىلاش شۇنچىلىك ئوڭاي، سىز تۆۋەندىكى بۇيرۇقلارنى تەرتىپ بويچە كىگۈزۈپلار Wine ئەڭ يېڭى نەشىرىنى Ubuntu غا قاچىلىيالايدسىز:

 

sudo add-apt-repository ppa:ubuntu-wine/ppa

sudo apt-get update

sudo apt-get install wine1.3

ئورگان بېتى: http://www.winehq.org/
