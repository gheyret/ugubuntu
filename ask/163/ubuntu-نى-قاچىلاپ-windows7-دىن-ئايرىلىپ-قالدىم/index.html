Ubuntu نى قاچىلاپ Windows7 دىن ئايرىلىپ قالدىم

0 نومۇر
كۆرسەتمىڭىز بويىچە Ubuntu نى مۇۋەپپىقيەتلىك قاچىلىۋالدىم. Ubuntu نى بارماق دېسكىغا ئويۇپ قاچىلىغان ئىدىم، قايسى رايونغا قاچىلاشنى تاللايدىغان باسقۇچقا كەلگەندە ئالدىنقى سوئالنىڭ جاۋابىدا دېيىلگىنىدەك قاچىلىماقچى بولغان دېسكا رايونىنى تاللاپ، ئۇنى ext4 ۋە swap قىلىپ ئىككىگە بۆلۈۋالدىم. ئاندىن ext4 قىلىپ بۆلگەن رايوننىڭ يەنە بىر تاللىشى بار ئىكەن،(غول مۇندەرىجە قىلىش-قىلماسلىقنى بېكىتىدىكەنمىز دەپ چۈشەندىم) ئۇنىڭدىن (/) نى تاللىدىم.(/etc,/var,/home دېگەندەك تاللاشلار بار ئىكەن). ئۇنىڭدىن كېيىن ئاستىدا Device for installing boot loader غۇ دەيمەن ،بىر تۈر بار ئىكەنتۇق، ئۇنى مەن يېتەكلەش رايونىنى نەگە قاچىلاشنى سورايدىكەن دەپ چۈشىنىپ Windows7 قاچىلانغان رايوننى تاللاپتىمەن. شۇنىڭ بىلەن قاچىلاش تاماملىنىپ كومپيۇتېرنى قايتا قوزغاتسام پەقەت Ubuntu لا قوزغىلالىدى.(Windows نىڭ تاللانمىسى چىقتى، لېكىن تاللىسام قوزغىلالمىدى) قارىغاندا Windows7 نىڭ boot manager نى Ubuntu نىڭ grub باسۇرىۋەتكەن ئوخشايدۇ. ھازىرقى سوئالىم: قانداق قىلسام ئىلگىرىكى Windows7 نى نورمال قوزغىلالايدىغان قىلالايمەن؟
