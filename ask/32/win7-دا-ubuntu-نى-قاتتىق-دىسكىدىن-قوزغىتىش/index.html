شۇنداق win7 قاچىلىغان C دىسكىدا XP غا ئوخشاش boot.init (مۇشۇنداققۇ دەيمەن ئىسىمدە يوق) دېگەن ھۆججەت يوق. بىۋاسيە قاتتىق دىسكىدىن قوزغۇتۇپ قاچىلاش سەل ئاۋارچىلىق، سىز بارماق دىسكىغا ياكى ئوپتىك دىسكىغا ئويۇپ شۇنىڭدىن قوزغۇتۇپ قاچىلىسىڭىز ئەڭ ياخشى.

ئەگەر win7 دا Ubuntu نىڭ ISO ھۆججىتىنى قاتىتىق دىسكىدىن قوزغۇتۇپ قاچىلىماقچى بولسىڭىز. Winrar ئارقىلىق ISO ھۆججىتىنى يېيىپ، caper مۇندەرىجىسى ئاستىدىكى initrd.lz ۋە vmlinuz ئىككى ھۆججەتنى C دىسكىنىڭ غول مۇندەرىجىسىگە قويۇڭ.

چۇشۇرگەن Ubuntu نىڭ ISO ھۆججىتىنىڭ ئىسمىنى ubuntu.iso غا ئۆزگەرتىپ، C دىسكىنىڭ غول مۇندەرىجىسىگە قويۇڭ.

http://u.115.com/file/dnc9pc9v بۇ يەردىن Grub چۈشۈرۈپ بارلىق ھۆججەتنى C دىسكىنىڭ غول مۇندەرىجىسىگە يىشىڭ.

ئەمدىن C دىسكىنى تەكشۈرسىڭىز، grub ئىچىدىكى ھۆججەتلەر ubuntu.iso،vmlinuz، initrd.lz، بۇلار تەل بولدىمۇ؟ ئۇنداق قايتا قوزغۇتۇڭ. بۇ ۋاقىتتا قوزغىتىش تىزىملىكىدىن grub نى تاللپ كىرىڭ ھەمدە قاچىلاشنى باشلىسىڭىز بولىدۇ.

ئەگەر قاچىلاش ئاخىرلاشقاندىن كىيىن قوزغىلىش تىزىملىكىدە win7 كۆرۈنمىسا تېرمىنالنى ئېچىپ . تۆۋەندىكى بۇيرۇقنى كىرگۈزۈڭ.
Code:
sudo update-grub
