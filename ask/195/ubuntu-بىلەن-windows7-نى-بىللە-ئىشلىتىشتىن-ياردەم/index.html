Ubuntu بىلەن Windows7 نى بىللە ئىشلىتىشتىن ياردەم

0 نومۇر
مەن يېقىندا خاتىرە كومپيۇتېردىن بىرنى سېتىۋالغان. كومپيۇتېردا ھازىر Windows7 نىڭ 32بىتلىق جا نەشرى قاچىلاقلىق. قاتتىق دېسكىدا 4رايون بار. ھازىر قاتتىق دېسكىنى بىرنەچچە رايونغا ئايرىپ، بىرىنچى رايونغا Windows7 نى، (ھازىر قاچىلاقلىق)، ئىككىنچى رايونغا Ubuntu نى قاچىلىماقچى، قالغان رايونلار باشقا ھۆججەتلەرنى ساقلاش ئۈچۈن قالدۇرىلىدۇ. (ئىككىلا سىستېمىغا ئورتاق ). رايونغا ئايرىش ھەققىدە ماتېرىيال كۆرسەم Ubuntu ئۈچۈن كەم دېگەندە ئىكككى رايون بولۇشى كېرەك (بىرسى ext3, يەنە بىرسى swap )دەپتۇ. كومپيۇتېرنىڭ فىزىكىلىق ئىچكى ساقلىغۇچى بىرقەدەر چوڭ (مەسىلەن 8GB) بولغان ئەھۋلدا Ubuntu بىلەن Windows7 نى بىللە ئىشلىتىپ، يۇقىرىدىدا بايان قىلىنغان مەقسەتلەرگە يېتىش ئۈچۈن رايونغا ئايرىش مەشغۇلاتىنى قانداق ئېلىپ بېرىش كېرەكلىكى ھەققىدە تەپسىلىيراق توختىلىپ ئۆتكەن بولسىڭىز.

سىز دېگەن مەقسەتكە ئەلۋەتتە يەتكىلى بولىدۇ. كومپيۇتېرنىڭ فىزىكىلىق ئىچكى ساقلىغۇچى 2g دىن چوڭ بولسا 64 بىتلىق سىستېما ئىشلىتىڭ. بولمىسا 4G لىق ۋە ئۇنىڭدىن چوڭ ئىچكى ساقلىغۇچ، windows, ubuntu نىڭ 32 بىتلىق سىستېمىسىدا ئارا 3G ئەتراپىدا ئىشلەيدۇ.

مەسىلەن كومپيۇتېرىڭىزنىڭ ھازىر 　C D E F تۆت رايونغا ئايرىدىڭىز دەيلى، E رايون 20G ئەتراپىدا دەپ پەرەز قىلايلى. سىز دەل مۇشۇ يەرگە ئۇبۇنتۇ قاچىلاپ، قوش سىستېما قىلىش نىيىتىگە يىتەلەيسىز.

قانداق قىلىپ دېگەندە، ئۇبۇنتۇ ئوپتىك دىسكىسىدىن ياكى بارماق دىسكىدىن كومپيۇتېرنى قوزغىتىپ، ئۇبۇنتۇ قاچىلاشنى باشلايسىز. ھەممە قەدەم باسقۇچ، iso نى بارماق دىسكىغا ئويۇش دېگەندەك مەزمۇنلار بىلوگدا بار (ئوپتىك دىسكىغا ئويۇشنى ئۆزىڭىزمۇ قىلالايسىز).

دىسكا ئايرىش باسقۇچىغا كەلگەندە، ئۇبۇنتۇدا دىسكا c d e دېگەندەك ئىسىملار بويىچە كۆرۈنمەيدۇ. sda1 sda 2 .. sda 5 دېگەندەك، قاتتىق دىسكىنىڭ بۆلىنىش رەت تەرتىپى بويىچە كۆرۈنىدۇ. سىز چوڭ كىچىكلىكىگە قاراپ، بايىقى مىسال قىلغان 20G لىق دىسكىنى تاپىسىز، ھەم شۇنىلا رايونغا ئايرىيسىز (باشقا دىسكىغا ئۇرۇنمايسىز). قاچىلاش تاماملىنىدۇ.
رايونغا ئايرىشتا ئەڭ ئاز دېگەندە سىز دېگەندەك ئىككى رايۇن بولسا بولىدۇ. بىراق بىرسى ext4 ئەمەس، ext4/. بۇ يەردىكى ext4 پەقەت دىسكىنىڭ شەكلى. بىز ھەرگىزمۇ windows دا دىسكا رايۇننى ntfs دىسكدا، fat32 دىسكا دېمەيمىز. شۇنىڭغا ئوخشاش،
ext4/ دېگىنىمىز، سىتېمىنىڭ غول مۇندەرىجىسى، شەكلى ext4.

تۆۋەندىكى رەسىملەرگە قاراپ بېقىڭ:


تۆۋەندىكى رەسىمدە، مەسىلەن سىز مەن بايا دېگەندەك دىسكا چوڭ كىچىكلىكىگە قاراپ، سىز ubuntu قاچىلىماقچى بولغان دىسكىنى تاپقاندىن كىيىن، شۇ دىسكىنى ئۆچۈرسىز.


تۆۋەنكى رەسىم swap رايۇنىنى ئايرىش باسقۇچى. باياقى ئۆچۈگەن رايۇننى ئايرىيسىز.




تۆۋەندىكى ئىككى رەسىمگە دىققەت قىلىسڭىز، ئەگەر دىسكا شەكلىنى 交换分区 قىلماي، ext4 قىلىسڭىز، ئاستىدا يەنە، boot/, ۋە home/, ۋە ياكى / دېگەندەك دىسكىنى ئاسىدىغان رايۇن كۆرۈنىدۇ. سىز ئەڭ ئاز رايۇنغا ئايرىماقچى بولسىڭىز، / نى تاللاپ، بايا ئۆچۈرگەن رايۇننىڭ بارلىق قىسمىنى / گە بەرسىڭىز بولىدۇ.

ئۇبۇنتۇ چۈشۈرۈشكە كەلسەك. ئىچكى ساقلىغۇچىڭىز 8G بولغاندىن كىيىن، 64بىتلىق سىستېما قاچىلىشىڭىز كېرەكلىكىنى، نېمىشقا مۇشۇنداق ئىكەنلىكىنى بىلىسىز. شۇڭا ھەر قاندا ۋاقىتتا desktop-amd64.iso دېگەننى چۈشۈرۈپ ئىشلەتسىڭىز بولىدۇ. بۇ يەردە نېمىشقا amd دەپ بولىدۇ، cpu رۇم intel تۇرسا دېيىشىڭىز مۇمكىن.
amd ئەڭ بۇرۇن 64بىتلىق CPU چىقارغاچقا، كېيىنچە 64بىتلىق دېتال، iso لار مۇشۇ ئىسىم بىلەن ئاتىلىپ قالغان خالاس. مىنىڭ CPU رۇم intel. مەنمۇ desktop-amd64.iso نى ئىشلىتىمەن.
