يېقىدىن بىر ئوبۇنتۇ ئىشلىتپ  خىلى نەرسىلەرنى ئۈگۈنۇپ قالدىم .بۇ جەرياندا ئاز بولمىغان قىيىنچىلىقلارغىمۇ يۇلۇقۇپ تۇردۇم.

chm شەكلىدىكى ھۆججەتلەرنى ۋىن دوستىغۇ بىۋاستە ئاچقىلى بۇلىدىكەن لىكىن  مەن ئوبۇنتۇ ئىشلەتكەندىن كېيىن chm شەكلىدىكى ئوبۇنتۇغا ئائىت بۇيرۇقلار توپلىمىنى چۈشۈرۈپ كۈرەي دىسەم ھېچ كۈرەلمىدىم شۇنىڭ بىلەن ئىزدەش ماتۇرلىرغا مۇراجەت قىلسام ئوبۇنتۇدا بۇنىڭغا مەخسۇس ئايرىم دىتال قاچىلايدىكەنمىز شۇنىڭ بىلەن دوستلارغا پايدىسى تېگىپ قالسا ئەجەپ ئەمەس دەپ بۇ يەرگىمۇ يوللاپ قۇيۇشنى لايىق تاپتىم.

چۈشۈرۈش ئورنى  : چۈشۈرۈش

ئۈزۈڭز خالىغان نۇسخىسىنى چۈشۈرۈش قاچىلسىڭز بۇلىدۇ.

 

بۇرۇق ئارقىلىق قاچىلاش ئۇسۇلى :

sudo apt-get install chmsee

 

 

ئەگەر مەسلە بايقىلىپ دىتال ئىچىلماي قالسا بۇ بۇيرۇقنى تىرمىنالدا ئىجرا قىلىڭ

cd /usr/lib
sudo ln -s xulrunner-1.9/libxul.so libxul.so
sudo ln -s xulrunner-1.9/libxpcom.so libxpcom.so
sudo ln -s xulrunner-1.9/libsqlite3.so libsqlite3.so
sudo ln -s xulrunner-1.9/libmozjs.so libmozjs.so
