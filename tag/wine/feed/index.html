Wine بولسا لىنۇكسدا Windows دېتاللىرىنى ئىشلىتىشتە ئىشلىتىلىدىغان دېتال. Windows دىكى نۇرغۇنلىغان مۇنەۋۋەر دېتاللارنى Wine ئارقىلىق لىنۇكسدا بىۋاسىتە ئىشلەتكىلى بولىدۇ. قارىماققا مەۋھۇم كومپيۇتېرغا ئوخشايدۇ، بىراق ئۇنىڭدىن كۆپ ئىلغار (ئىسىڭلاردا بولسۇن ھەممە دېتالنى Wine ئارقىلىق لىنۇكسقا قاچىلىيالمايمىز).

بۇ قېتىم ئېلان قىلىنغان Wine 1.3.23 دا ئىلگىرىكى نۇرغۇن Bug لار تۈزىتىلىپتۇ، ماسلىشىشچانلىقى (兼容性) ياخشىلىنىپتۇ، بىر سىناپ بېقىشقا ئەرزىگۈدەك.

ئورگان بېتى:http://www.winehq.org

چۈشۈرۈش ئادرېسى:http://www.winehq.org/download

Wine نى بۇ نەشرى PPA گە قوشۇلۇشتىن بورۇن كونا نەشرىنى PPA مەنبەسىدىن قاچىلاش بۇيرۇقى:

sudo add-apt-repository ppa:ubuntu-wine/ppa
sudo apt-get update
sudo apt-get install wine1.3

ئۇبۇنتۇدا PS cs4 نى ئىشلىتىش

QQ-for -wine نىڭ deb بولىقىنى چۈشۈرۈش ئادرېسى: http://www.longene.org/download/qq-for-wine_20110425.deb

ئىشلەتكەندە چۈشۈرگەن deb بولىقى بار مۇندەرىجىسىگە كىرىپ تۇرۇپ تۆۋەندىكى بۇيرۇق بىلەن قاچىلاڭ، ئامال بار قوش چېكىپ قاچىلىماڭ.

sudo dpkg -i qq-for-wine_20110425.deb

ئەگەر ئۈستەل يۈزىگە qq.exe سىن بەلگىسى چىقمىسا تېرمىنالدا /usr/local/wineapp/qq/ مۇندەرىجىسىگە كىرىپ qq.sh بىر يۈرگۈزۈۋېتىمىز.
ئۆچۈرگەندە sudo dpkg -r qq-for-wine.deb ياكى sudo dpkg -P qq-for-wine.deb بۇيرۇق ئارقىلىق ئۆچۈرىمىز.

بۇ خىل ئۇسۇل بىلەن ئۇبۇنتۇغا قاچىلىغان qq بولسا، qq 2010 نىڭ ھەممە ئىقتىدارىنى دېگۈدەك ساقلاپ قالغان، بىراق يەنىلا يەنىلا لىنۇكس ئۆزى ئۈچۈن مەخسۇس ئىشلەنگەن qq بولمىغانلىقتىن، سۈرەت سەل ئاستا، بىراق wine غا بىرمۇ بىر قول سېلىپ قاچىلىغانغا قارىغاندا كۆپ قۇلايلىق.

  • WINE قاچىلاش

Wine نىڭ ئەڭ يېڭى نۇسخىسى بولغان Wine 1.3 نى PPA مەنبەسىدىن چۈشۈرىمىز؛

قاچىلاش بۇيرۇقى

www.uyghur-archive.com/ugubuntuputer:~$sudo add-apt-repository ppa:ubuntu-wine/ppa
www.uyghur-archive.com/ugubuntuputer:~$sudo apt-get update
www.uyghur-archive.com/ugubuntuputer:~$sudo apt-get install wine1.3
  • PlayOnLinux قاچىلاش

- Ubuntu 10.10 Maverick Meerkat:

www.uyghur-archive.com/ugubuntuputer:~$wget -q ”http://deb.playonlinux.com/public.gpg“ -O – | sudo apt-key add -
www.uyghur-archive.com/ugubuntuputer:~$sudo wget http://deb.playonlinux.com/playonlinux_maverick.list -O /etc/apt/sources.list.d/playonlinux.list
www.uyghur-archive.com/ugubuntuputer:~$sudo apt-get update
www.uyghur-archive.com/ugubuntuputer:~$sudo apt-get install playonlinux

- Ubuntu 10.04 Lucid Lynx:

www.uyghur-archive.com/ugubuntuputer:~$wget -q ”http://deb.playonlinux.com/public.gpg“ -O – | sudo apt-key add -
www.uyghur-archive.com/ugubuntuputer:~$sudo wget http://deb.playonlinux.com/playonlinux_lucid.list -O /etc/apt/sources.list.d/playonlinux.list
www.uyghur-archive.com/ugubuntuputer:~$sudo apt-get update
www.uyghur-archive.com/ugubuntuputer:~$sudo apt-get install playonlinux

دىققەت:

    PlayOnLinux نى تۇنجى قاچىلىغاندا توردىن يېڭىلىنىش ئەھۋالىنى تەكشۈرىدۇ ھەمدە فونت چۈشۈرۈپ ئاپتوماتىك قاچىلايدۇ.

  • PS cs4 قاچىلاش

يۇقىرىقىلارنى قاچىلاش تاماملانغاندىن كىيىن Application>>Games دىن PlayOnLinux نى قوزغىتىمىز. ئاندىن Install دېگەن كۇنۇپكىنى بېسىپ PS cs4 نى قاچىلاشنى باشلايمىز.

رەسىمدىكى رەقەم تەرىپى بويىچە قاچىلاشنى داۋام ئىتىمىز.

كۆزنەكتىن Forward نى تاللاپ قاچىلاشنى داۋام ئىتىمىز.

بۇ جايدا PlayOnLinux توردىن كېرەكلىك بولغان Wine سەپلىمىسىنى چۈشۈرىدۇ، سەل ساقلاشقا توغرا كېلىدۇ.

Browse نى بېسىپ قاچىلىماقچى بولغان PS cs4 نىڭ exe ھۆججىتىنى تاللايمىز، ئاندىن Forward نى بېسىپ داۋاملاشتۇرىمىز.

ئەمدى PS قاچىلىنىشقا باشلايدۇ، PS قاچىلاشنى يەرشارىدىكى ھەممە ئادەم بىلسە كېرەك. بىراق دىققەت قىلىدىغان ئىش شۇكى تاللىغان PS قانچە كىچىك ئىخچام بولسا شۇنچە ياخشى، ئەگەر تاپالمىغانلار QQ ئېلخەت ئادرېسىنى قالدۇرۇپ قويسا مەن سىناق قىلىپ قاچىلىغان PS cs4 ئەۋەتىپ بېرىمەن.PS cs4 نى قوزغىتىش ئۇسۇلى ئىككى خىل بولۇپ PlayOnLinux تىزىملىكىدىن ياكى Wine تىزىملىكىدىن قوزغاتساق بولىدۇ.

قاراڭلار PS قوزغىلىۋاتىدۇ.

قاچىلىغان PS ۋىندوۋس نۇسخىسى بولغانلىقتىن سۈرئەت جەھەتتە يەنىلا ئاستىراق، بىراق يوقتىن ياخشى دە.
