ئۇبۇنتۇ ئىشلىتىپ باققانلار بىلىدۇ، قاچىلاش داۋامىدا كومپيۇتېرنى ئۇبۇنتۇنىڭ liveCD قوزغاتساق، ھازىر قاچىلاش ۋە ھازىر ئىشلىتىش دەپ ئىككى تاللاش كۆرۈنىدۇ. قاچىلاشنى بېسىپ ئۇبۇنتۇنى قاتتىق دىسكىغا قاچىلاش مەشغۇلاتىنى قىلىمىز. ھازىر ئىشلىتىشنى بېسىپ liveCD دىكى ئۇبۇنتۇنى ئىشلىمەيمىز. ئۇبۇنتۇنى بارماق دىسكىغا قاچىلاش ۋە بارماق دىسكىدىن قوزغىتىپ قاچىلاشمۇ مۇشۇنىڭغا ئوخشاش.

ئۇبۇنتۇنى بارماق دىسكىغا قاچىلاش دېگىنىمىز ئۇبۇنتۇ iso سىنى بارماق دىسكىغا (يان دىسكىمۇ بولىدۇ، ھەتتا sd كارتىمۇ بولىدۇ) ئويۇپ، كومپيۇتېرنى بارماق دىسكىدىن قوزغىلىدىغان قىلىپ تەڭشەيمىز. ئاندىن ھازىر ئىشلىتىش (try to use مۇشۇنداقراق) بېسىپ ئىشلەتسەكلا بولىدۇ.
بۇ خىل ئۇسۇلنىڭ ئارتۇقچىلىقى: خالىغان جايدا خالىغان كومپيۇتېردا بارماق دىسكىمىزدىكى ئۇبۇنتۇنى ئىشلىتەلەيمىز. بىراق سۈرەت ئازراق ئاستا بۆلىشى، قوزغاتقۇچلار ئىشلىمەسلىكى مۇمكىن.
بارماق دىسكىدىن قوزغىتىپ قاچىلاش دېگىنىمىز ئۇبۇنتۇ iso سىنى بارماق دىسكىغا (يان دىسكىمۇ بولىدۇ، ھەتتا sd كارتىمۇ بولىدۇ) ئويۇپ، كومپيۇتېرنى بارماق دىسكىدىن قوزغىلىدىغان قىلىپ تەڭشەيمىز. ئاندىن ھازىر قاچىلاش (Install now) بېسىپ ئۇبۇنتۇنى liveCD دىن قاچىلىغانغا ئوخشاش قاچىلاش مەشغۇلاتى قىلىمىز.
بۇ خىل ئۇسۇلنىڭ ئارتۇقچىلىقى: CD/DVD دىسكىسى ئالىدىغان پۇلىڭىزنى تېجەپ قالىمىز ئەلۋەتتە. ھى…ھى…
بۇ يەردىكى مەسىلە ئۇبۇنتۇ iso سىنى قانداق قىلىپ بارماق دىسكىغا توغرا ئۇيۇش. ئەلۋەتتە iso نى بارماق دىسكىغا كۆچۈرۈپ قويساق بولمايدۇ. مەن تۆۋەندە Windows شارائىتىدا ئۆزۈم كۆپ ئىشلىتىدىغان ئۇسۇل بىلەن، ئۇبۇنتۇ ئورگان بېتى تەۋسىيە قىلغان ئىككى خىل ئۇسۇلنى تونۇشتۇرۇپ ئۆتەي.

  • UltraISO

UltraISO نى قاچىلاپ ئالدىن تەييارلىۋالغان ئۇبۇنتۇ iso سىنى UltraISO ئارقىلىق ئاچىمىز. ئاندىن تۆۋەندىكى رەسىمدىكىدەك 启动光盘–写入硬盘映像 نى باسىمىز. UltraISO ھەقلىق دېتال بولغاچقا xxx نۇسخىسىنى ئۆزۈڭلار بەيدۇ قىلىڭلار. ھى…ھى…

ئېچىلغان كۆزنەكتىن ：便捷启动–写入新的驱动器引导区–Syslinux نى تاللايمىز. بۇ باسقۇچ بولمىسا بارماق دىسكىغا ئويۇلغان ئۇبۇنتۇ boot missing دەپ قوزغالمايدۇ جۇمۇ.


ئەمدى ئۆزىمىز ئىشلەتمەكچى بولغان بارماق دىسكىنى (يان دىسكا) تاللاپ رەسىمدىكىدەك مەشغۇلات قىلساقلا بولدى.

  • Universal USB Installer (ئۇبۇنتۇ ئورگان بېتى تەۋسىيە قىلغان ئۇسۇل)

ئالدى بىلەن Universal USB Installer نىڭ ئەڭ يېڭى نۇسخىسىنى چۈشۈرۈپ قوزغىتىمىز. ھەم رەسىمدىكى 1 دېگەن ئورۇندىن ئۆزىمىز ئىشلەتمەكچى بولغان ئۇبۇنتۇ نۇسخىسىنى، 2 دېگەن ئورۇندىن تەييارلىۋالغان ئۇبۇنتۇ iso سىنى تاللايمىز.

ئەمدى تۆۋەندىكى رەسىمدىكى 1 نىڭ سول تەرىپىدىن ئۆزىمىز ئىشلەتمەكچى بولغان بارماق دىسكىنى (يان دىسكا) تاللايمىز، 1 نىڭ ئوڭ تەرىپىنى تاللاپ قويساق ئۇيۇشتىن بۇرۇن بارماق دىسكا (يان دىسكا) فورماتلىنىدۇ، ئېھتىيات قىلىڭلار. 2 دېگەن ئورۇندىن ئىشلەتمەكچى بولغان بوشلۇقنى تاللايمىز.

ئىككى خىل ئۇسۇلنى تونۇشتۇرۇپ بولدۇم، ئۇبۇنتۇنى بارماق دىسكىغا قاچىلاپ ئىلىپ يۈرسەڭلارمۇ ۋە بارماق دىسكىدىن قوزغىتىپ قاچىلاپ پۇل تىجىسەڭلارمۇ بولىدۇ (بارماق دىسكىسى بارلار).

  • چۈشۈرۈلگەن iso نى نۇر دىسكىغا ئويىمىز (بارماق دىسكىغا ئويسىمۇ بولىدۇ).
  • كومپيۇتېرنىڭ BIOS نى بەلگىلەيمىز. يەنى ئۇنى CD_ROM دىن قوزغىلىدىغان قىلىپ تەڭشەيمىز. ئاندىن تەييارلىۋالغان CD نى سېلىپ كومپيۇتېرنى قايتا قوزغىتىمىز (ئەگەر بارماق دىسكىغا ئىشلەتكەن بولساق كومپيۇتېرنى USB دىن قوزغىلىدىغان قىلىپ كومپيۇتېرنى قايتا قوزغىتىمىز ).
  • رەسىمىدىكى كۆزنەك كۆرۈنگەندىن كېيىن ئەڭ ئۈستىدىكىنى بېسىپ ئۇبۇنتۇنى قاچىلىمايمۇ ئىشلىتەلەيمىز. مەن بۇ يەردە ئاستىدىكى تاللاپ داۋاملىق قاچىلىدىم.

ئالدى بىلەن ISO باشقۇرىدىغان يۇمتالدىن بىرنى چۈشۈرىمىز. مەن UltraISO تەۋسىيە قىلىمەن. UltraISO نى كومپيۇتېرغا ئورناتقاندىن كىيىن ئۇبۇنتۇنىڭ ئورگان بېتى، ياكى مەن چۈشۈرۈش بېتىدە تەمىنلىگەن ئادرېستىن ئۇبۇنتۇنىڭ ISO ھۆججىتىنى چۈشۈۋالىمىز. ئاندىن ئۇنىڭ خالىغان ئورۇنغا قۇيىمىز (ئەڭ ياخشىسى C دىسكىغا قويماڭ). مەن تۆۋەندىكى رەسىمدە ISO نى D دىسكىنىڭ غول مۇندەرىجىسىگە قويدۇم.

ئاندىن UltraISO ئارقىلىق ISO ھۆججىتىنى ئېچىپ wubi.exe چىقىرىمىز.

چىقىرىۋالغان wubi.exe چوقۇم ئۇبۇنتۇ ISO ھۆججىتىنى بىلەن بىر مۇندەرىجىدە بۆلىشى كېرەك.

wubi.exe نى ئىجرا قىلىپ قاچىلاش مەشغۇلاتىنى باشلايمىز. رەسىمدىكى كۆزنەكتىن قاچىلىماقچى بولغان ئورۇننى، بۆلۈپ بەرمەكچى بولغان بوشلۇقنى، سىستېما تىلىنى، پارولنى بېكىتىمىز.

يۇقىرىقى مەشغۇلات توغرا بولسا تۆۋەندىكى رەسىمدىكىدەك كۆزنەك چىقىدۇ.

بىرئاز ساقلاپ تۆۋەندىكى كۆزنەك چىققاندا كومپيۇتېرنى قايتا قوزغىتىمىز.

قايتا قوزغىتىش جەريانىدا شۇنى بايقايمىز، يەنى قارا ئېكراندا ubuntu دېگەن تاللاش كۆزنىكى كۆرۈنىدۇ. شۇنى تاللاپ، ئاندىن چىققان تىزىملىكنىڭ بىرىنچىسىنى تاللاپ بەش مىنۇتتەك تەخىر قىلساق قاچىلاش پۈتۈنلەي تاماملىنىدۇ.
wubi قاچىلىغان ئۇبۇنتۇنى ئۆچۈرۈشمۇ ناھايىتى ئاسان، يەنى Windows دا ئادەتتىكى يۇمتالنى ئۆچۈرگەنگە ئوخشاش.
