Wine بولسا لىنۇكسدا Windows دېتاللىرىنى ئىشلىتىشتە ئىشلىتىلىدىغان دېتال. Windows دىكى نۇرغۇنلىغان مۇنەۋۋەر دېتاللارنى Wine ئارقىلىق لىنۇكسدا بىۋاسىتە ئىشلەتكىلى بولىدۇ. قارىماققا مەۋھۇم كومپيۇتېرغا ئوخشايدۇ، بىراق ئۇنىڭدىن كۆپ ئىلغار (ئىسىڭلاردا بولسۇن ھەممە دېتالنى Wine ئارقىلىق لىنۇكسقا قاچىلىيالمايمىز).

بۇ قېتىم ئېلان قىلىنغان Wine 1.3.23 دا ئىلگىرىكى نۇرغۇن Bug لار تۈزىتىلىپتۇ، ماسلىشىشچانلىقى (兼容性) ياخشىلىنىپتۇ، بىر سىناپ بېقىشقا ئەرزىگۈدەك.

ئورگان بېتى:http://www.winehq.org

چۈشۈرۈش ئادرېسى:http://www.winehq.org/download

Wine نى بۇ نەشرى PPA گە قوشۇلۇشتىن بورۇن كونا نەشرىنى PPA مەنبەسىدىن قاچىلاش بۇيرۇقى:

sudo add-apt-repository ppa:ubuntu-wine/ppa
sudo apt-get update
sudo apt-get install wine1.3

Windows دېگەن ئاتالغۇ كۆپىنچە كىشىلەرنىڭ ئىدىيىسىدە كومپيۇتېر دېگەن ئاتالغۇ بىلەن بىرلىشىپ كەتكەن، ھەتتا Windows بولمىسا كومپيۇتېرنى ئىشلەتكىلى بولمايدۇ دەپ ئويلايدىغانلارمۇ ئاز ئەمەس. بۇنىڭ ئاساسلىق سەۋەبى بىزگە ئۇچرىغان كومپيۇتېرلارنىڭ كۆپىنچىسىگە Windows مەشغۇلات سىستېمىسىنىڭ قاچىلانغانلىقىدۇر.ئۇنداقتا كومپيۇتېر ئىشلەتكۈچىلىرىگە Windows مەشغۇلات سىستېمىسىنى قاچىلاشتىن باشقا تاللاش يوقمۇ ؟
ئەمەلىيەتتە ھەرگىز ئۇنداق ئەمەس. كومپيۇتېر تورىدىن ياكى مەخسۇس مەشغۇلات سىستېمىسى تونۇشتۇرۇلغان كىتابلاردىن ئىزدەيدىغان بولساق Dos, Windows, Unix, Linux,Minix, MAC OS……قاتارلىق ئوندىن ئارتۇق مەشغۇلات سىستېمىسىنىڭ ئىسمىنى تاپالايمىز. ئەگەر يېڭىدىن ئىشلىنىۋاتقان، ئىشلىنىش ئالدىدا تۇرغان ۋە ئىشلىتىلىشتىن قالغانلىرىنى قوشىدىغان بولساق تېخىمۇ كۆپ. بىراق كومپيۇتېر بىزگە كىرىپ ئىشلىتىلىشكە باشلىغاندىلا مىكروسوفت شىركىتىنىڭ Dos مەشغۇلات سىستېمىسى ئارقىلىق كىرگەن ھەم مىكروسوفتنىڭ يۇمشاق دېتال تەرەققىياتىنى بويلاپ كەڭ تۈردە ئومۇملاشقان بولغاچقا بىزگە Windows دېگەن ئاتالغۇ بەكلا چوڭقۇر ئورناپ كەتكەن. مەسىلەن كومپيۇتېر بىلەن دەسلەپ تونۇشقانلارنىڭ كۆپىنچىسى Windows 3.2, Windows 95, Windows 98, Windows ME, Windows 2000, Windows XP قاتارلىقلارنى ئىشلىتىپ كۆرگەن ،تېخى يېڭىدىن كومپيۇتېرنى ئىشلىتىشكە باشلىغانلار بولسىمۇ بىۋاسىتە Windows 2000 ياكى Windows XP بىلەن ئۇچرىشىدۇ. چۈنكى Windows مەشغۇلات سىستېمىسىنىڭ ئىقتىدارى ئىنتايىن كۈچلۈك، ئىشلىتىش ناھايىتى قۇلايلىق. بىراق بۇ باشقا مەشغۇلات سىستېمىلىرى Windows قا يەتمەيدۇ دېگىلى بولمايدۇ. ھەرخىل مەشغۇلات سىستېمىسىنىڭ ئۆزگىچە ئالاھىدىلىكلىرى بار. مەسىلەن : MAC OS نىڭ كۆزنەكلىك كۆرۈنۈش ۋە مۇقىملىق جەھەتتىكى ئىقتىدارىنى پۈتۈن دۇنيا ئېتىراپ قىلىدۇ. بىراق بۇ سىستېما پەقەت ئالما ماركىلىق كومپيۇتېرلاردىلا ئەڭ ياخشى ئىشلەيدۇ. Unix, Linux مەشغۇلات سىستېمىلىرىنىڭ كومپيۇتېر تورى، كومپيۇتېر بىخەتەرلىكى ۋە سىستېما مۇقىملىقى جەھەتتىكى ئىقتىدارى ئىنتايىن كۈچلۈك… گەرچە PC كومپيۇتېر ساھەسىدە Windows ئاساسلىق ئورۇننى ئىگىلەۋاتقان بولسىمۇ، بىراق ئۆزگىچە ئىقتىدارى بىلەن يەنىلا خېلى كۆپ ساھەنى كونترول قىلىپ تۇرۇۋاتقان ۋە ئۈزلۈكسىز تەرەققىي قىلىۋاتقىنى يەنىلا Linux مەشغۇلات سىستېمىسى. Linux نۆۋەتتىكى كومپيۇتېر مەشغۇلات سىستېمىسى ساھەسىدىكى بىر قىزىق نۇقتا ھېسابلىنىدۇ. دەسلىپىدە بۇ مەشغۇلات سىستېمىسى قاچىلاش جەھەتتىكى بەزىبىر يېتەرسىزلىكى ، تەشۋىقاتنىڭ ياخشى بولماسلىقى ۋە باشقا قىسمەن سەۋەبلەر تۈپەيلىدىن كەڭ ئىشلىتىشكە ئېرىشەلمەي كەلگەن بولسىمۇ ، بىراق ئالدىنقى نەچچە يىل ئىچىدىكى تەرەققىيات نەتىجىسىدە ھەرقايسى جەھەتتىكى ئىقتىدارى مۇكەممەللىشىپ بىر قېتىملىق Linux دولقۇنى قوزغالدى. بۇ مىكروسوفت شىركىتىگىمۇ خېلى زور بېسىم ئېلىپ كەلدى.
مىنىڭ بۇ يەردە دېمەكچى بولغىنىم ھەرگىزمۇ Linux مەشغۇلات سىستېمىسى كۆككە كۆتۈرۈپ Windows مەشغۇلات سىستېمىسىنى چۈشۈرۈش ئەمەس، بەلكى كۆپچىلىككە تېخىمۇ بىخەتەر تېخىمۇ ئەرزان بولغان Linux مەشغۇلات سىستېمىسىنى تونۇشتۇرۇشتۇر خالاس. ئەگەر سىز Linux مەشغۇلات سىستېمىسىنى ئىشلىتىشنى ئۆگەنمەكچى بولسىڭىز ، Ubuntu Linux مەشغۇلات سىستېمىسى سىزگە ئەڭ ياخشى بوسۇغا بولالايدۇ.
