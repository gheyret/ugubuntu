]]> https://www.uyghur-archive.com/ugubuntu/ubuntu-unity-android-skin.html/feed 0 ئۇبۇنتۇ 11.04 سۈكۈت ئۇسلۇبىنى يېڭىچە ئۈنۈمگە ئىگە قىلىش https://www.uyghur-archive.com/ugubuntu/nautilus-elementary-ambiance-theme-in-ubuntu11-04.html https://www.uyghur-archive.com/ugubuntu/nautilus-elementary-ambiance-theme-in-ubuntu11-04.html#comments Sat, 21 May 2011 00:37:35 +0000 Bakhtiyar https://www.uyghur-archive.com/ugubuntu/?p=1405 تۆۋەندىكى مىنىڭ ئۇبۇنتۇرۇمنىڭ ئۈستەل يۈزىنى بىر كۆرۈپ بېقىڭلار. سىلەرنىڭ بىلەن قانداق ئوخشىماسلىقى بار ئىكەن. تېخىمۇ چىڭ قارىساڭلار بىلەلەيسىلەر (ئوڭ تەرىپىدىكى conky دېتالى، بۇنى كىيىن تونۇشتۇرىمەن).


ئەمەلىيەتتە مىنىڭ ئىشلەتكىنىممۇ ئۇبۇنتۇ 11.04 سۈكۈت ئۇسلۇبى Ambiance. زەن سېلىپ قارىساڭلار، ئوڭ تەرەپتىكى مۇندەرىجە، دىسكا ئورنىدا ئۆزگىرىش بولغان، سول يۇقىرى تەرىپىگە، سىن بەلگىنى ئەسلىي ھالەتتە، تىزىملىك ھالىتىدە كۆرسىتىدىغان، ئۈچ دانە كىچىك كۇنۇپكا قوشۇلغان. كۆزنەك رەڭگى پۈتۈنلەي قارا قوڭۇر رەڭگە ئۆزگىرىپ تېخىمۇ كۆركەملەشكەن.

ئەمەلىيەتتە بۇ ئۈنۈمگە ئېرىشىش بەكمۇ ئاسان، Nautilus Elementary نى قاچىلايمىز.

sudo add-apt-repository ppa:am-monkeyd/nautilus-elementary-ppa
sudo apt-get update
sudo apt-get dist-upgrade
nautilus -q

بۇنى قاچىلىغاندىن كىيىن، كۆزنەك رەڭگىدىن باشقا ھەممە نەرسە يۇقىرىدىكى رەسىمدىكىدەك كۆرۈنىدۇ. ئەمدى تۆۋەندىكى ئۇلىنىشتىن Ambiance ئۇسلۇبىنىڭ ئازراق قىستۇرمىسىنى چۈشۈرىمىز.
http://fc03.deviantart.net/fs71/f/2011/138/1/5/unfinished_work_by_simplygreat-d3golt2.zip

چۈشۈرگەن zip ھۆججىتىنى يېشىم، ئىچىدىكى Ambiance ھۆججەت قىسقۇچىنى، ئۆزىمىزنىڭ home مۇندەرىجىسى ئاستىدىكى themes. مۇندەرىجىسىگە ئەكىرىپ قويساقلا بولدى (يەنىthemes./~ غا).

ئۇبۇنتۇ 11.04 كە untiy ئىشلىتىلگەن بىلەن، ئەمەلىيەتتە gnome 2 شۇ. تۆۋەندە ئۆزۈم ئىشلىتىپ باققان ئىككى ئۇسلۇبنىڭ قاچىلاش بۇيرۇقىنى تەمىنلەي، ئىشلىتىپ باقارسىلەر.

  • Equinox Evolution ئۇسلۇبى
sudo add-apt-repository ppa:tiheum/equinox
sudo apt-get update
sudo apt-get install gtk2-engines-equinox equinox-theme
  • Orta ئۇسلۇبى
sudo add-apt-repository ppa:nikount/orta-desktop
sudo apt-get update
sudo apt-get install orta-theme

Macubuntu ئۇسلۇبىنى ئۇبۇنتۇغا قاچىلاش ئارقىلىق ئۇبۇنتۇنى MacOs قا ئوخشاش قىلىۋەتكىلى بولىدۇ. كىشىگە تۇيۇقسىزلا ھەقسىز MacOs سىستېمىسىغا ئېرىشكەندەك تۇيغۇ بېرىدۇ. مەن ئۇبۇنتۇنىڭ ئەسلى سۈكۈت ئۇسلۇبنى ياقتۇرساممۇ، كۆپچىلىككە ئۇبۇنتۇغا قانداق قىلىپ بۇ ئۇسلۇبنى قاچىلاپ ئالما چاپىنىنى كىيدۈرۈشنى بىر سۆزلەپ ئۆتەي.
قېنى باشتا Macubuntu ئۇسلۇبىنى قاچىلىغاندىن كېيىنكى ئۇبۇنتۇ ئارايۈزىنى بىر كۆرۈپ باقايلى:





Macubuntu ئۇسلۇب تۈرىنىڭ ئورگان بېتى: http://sourceforge.net/projects/macbuntu
ئالدى بىلەن ئىشلىتىۋاتقان ئۇبۇنتۇ نۇسخىسىغا ئاساسەن Macubuntu ئۇسلۇبنى چۈشۈرىمىز.
ئۇبۇنتۇ 10.04 Macubuntu ئۇسلۇبىنى چۈشۈرۈش ئادرېسى:

http://sourceforge.net/projects/macbuntu/files/macbuntu-10.04

ئۇبۇنتۇ 10.10 Macubuntu ئۇسلۇبىنى چۈشۈرۈش ئادرېسى:

http://sourceforge.net/projects/macbuntu/files/macbuntu-10.10

قاچىلاش ئۇسۇلى:
باشتا ئۆزىمىز چۈشۈرگەن Macbuntu-10.04.tar.gz ياكى Macbuntu-10.10.tar.gz يېشىمىز. ئاندىن تېرمىنالدا cd بۇيرۇقى بىلەن يېشىلگەن Macbuntu-10.04 ياكى Macbuntu-10.10 ھۆججەت قىسقۇچى مۇندەرىجىسىگە كىرىپ، تۆۋەندىكى بۇيرۇق بىلەن قاچىلايمىز.
