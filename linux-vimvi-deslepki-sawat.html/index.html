ئېلان قىلىنغىلى679 كۈن بولدى / يۇمشاق دېتال تەۋسىيىسى / 62 پارچە ئىنكاس چۈشتى / كۆرۈلگىنى 1,166+

لىنۇكس مەشغۇلات سىستېمىسىدا ئەڭ كۆپ قوللىنىلىدىغان ھۆججەت تەھرىرلىگۈچ بولسا vi/vim بولۇپ، بۇنى windows دىكى txt . ھۆججىتى بىلەن ئوخشاش دېيىشكە بولىدۇ (مېنىڭچە vi/vim بولسا txt . دىن كۆپ ئىلغار). سىز Linux ئىشلىتىدىكەنسىز بۇ ئىككى خىل تەھرىرلىگۈچنى چوقۇم ئىشلىتەلىشىڭىز كېرەك. مەن تۆۋەندە ئۆزۈم بىرقەدەر ياقتۇرىدىغان vim تەھرىرلىگۈچى ئارقىلىق كۆپلۈككە Linux مەشغۇلات سىستېمىسىدا ھۆججەت تەھرىرلىگۈچنى قانداق ئىشلىتىشنى تونۇشتۇرۇپ ئۆتەي. vim نى vi نىڭ نەشرى كۆتۈرۈلگەن نۇسخىسى دەپ چۈشىنىشكە بولىدۇ.

ئالدى بىلەن vim نى قاچىلايمىز. (ئادەتتە ئۇبۇنتۇدا vi سىستېما بىلەن بىرگە قاچىلاقلىق).

www.uyghur-archive.com/ugubuntuputer:~$sudo apt-get install vim

تېرمىنالغا كېرىپ vim höjjetismi.shekli شەكلىدە ئۆزىمىز تۇرۇۋاتقان مۇندەرىجە ئاستىدا ھۆججەت تەھرىرلەش ئېلىپ بارالايمىز. مەسىلەن مەن ئۈستەل يۈزىگە بىر txt. قۇرۇپ ئۇنىڭغا !I Love Linux دەپ يازاي.

ئالدى بىلەن ئۈستەل يۈزى مۇندەرىجىسىگە كىرىش

www.uyghur-archive.com/ugubuntuputer:~$cd /home/name/Desktop

ئاندىن abc.txt دىن بىرنى قۇرۇپ ئۇنى تەھرىرلەيمىز.

www.uyghur-archive.com/ugubuntuputer:~$vim abc.txt

ئاندىن رەسىمدىكىدەك رامكا كۆرۈنىدۇ. بىراق سىز بىۋاسىتە كۇنۇپكا تاختىسى ئارقىلىق ئۇنىڭغا خەت كىرگۈزسىڭىز، ھېچنېمە كىرگۈزەلمەيمىز. شۇڭا ئالدى بىلەن a ياكى i كۇنۇپكىسىنى بىر قېتىم بېسىپ ئاندىن ئاندىن كىرگۈزمەكچى بولغان مەزمۇننى كىرگۈزىمىز.

سىز بۇ يەردە نېمىشقا ئالدى بىلەن a كۇنۇپكىسىنى بىر قېتىم بېسىپ ئاندىن كىرگۈزىمىز دەپ سورىشىڭىز مۇمكىن. مىنىڭە بۇ بەك ياخشى لايىھە بولۇپ ئۆزىڭىز ئىشلىتىش جەريانىدا ھېس قىلالايسىز. قېنى بىز كىرگۈزۈش ئۇسۇلى جەدۋىلىنى كۆرۈپ باقايلى:
a———————————————–نۇر بەلگىنىڭ كەينىدىن باشلاپ كىرگۈزۈش
A———————————————- قۇرنىڭ ئەڭ كەينىدىن باشلاپ كىرگۈزۈش
i———————————————–نۇر بەلگىنىڭ ئۈستىدىن باشلاپ كىرگۈزۈش
I———————————————–قۇرنىڭ ئەڭ ئالدىدىن باشلاپ كىرگۈزۈش
o———————————————-نۇر بەلگە كەينىدىن يېڭى قۇر ھاسىل قىلىپ كىرگۈزۈش
O———————————————-نۇر بەلگە ئالدىدىن يېڭى قۇر ھاسىل قىلىپ كىرگۈزۈش
دىققەت:تۆۋەندىكى كۇنۇپكىلارنى ئالماشتۇرۇپ ئىشلەتكەندە چوقۇم Esc كۇنۇپكىسىنى بىر قېتىم بېسىپ ئاندىن يېڭى كىرگۈزۈش ئۇسۇلى كۇنۇپكىسىنى بېسىشىمىز كېرەك. مەسىلەن مەن a كۇنۇپكىسىنى بېسىش ئارقىلىق تەھرىرلەشنى باشلىغان بولسام ھەمدى نۇر بەلگە كەينىدىن يېڭى قۇر ھاسىل قىلىپ كىرگۈزمەكچى بولغان بولسام ئالدى بىلەن Esc كۇنۇپكىسىدىن بىر قېتىم بېسىپ ئاندىن o نى بېسىپ تەھرىرلەشنى داۋاملاشتۇرۇشۇم كېرەك.
ئەگەر مەن يازغان مەزمۇننى ساقلاپ چېكىنمەكچى بولسام Esc كۇنۇپكىسىنى بىر قېتىم بېسىپ ئاندىن :+Ctrl نى باسساق رامكىنىڭ ئەڭ ئاستىدا كىرگۈزۈش كۆزنىكى پەيدا بولىدۇ، ئاندىن ئۇ يەرگە wq نى كىرگۈزۈپ جەزملەشتۈرسە كابۇلدا.

نېمىشقا wq كىرگۈزۈپ چېكىنىشنىڭ سەۋەبى يۇقىرىدا دەپ ئۆتكەن a بېسىپ ئاندىن تەھرىرلەشكە ئوخشىشىپ قالىدۇ. تەھرىرلىگۈچنىڭ قوشۇمچە ئىقتىدار جەدۋىلى تۆۋەندىكىچە:
q————————————————-مەزمۇننى ساقلىماي تەھرىرلەشتىن ۋاز كېچىش
!q————————————————مەزمۇننى ساقلىماي تەھرىرلەشتىن مەجبۇرى ۋاز كېچىش
wq———————————————–مەزمۇننى ساقلاش ھەمدە تەھرىرلەشتىن چېكىنىش
e————————————————-يېڭىدىن قۇرۇش
بىراق سىستېمىنى خەنزۇچە قىلغاندا Application–>Accessories دىكى Gvim نىڭ دىئالوگ رامكىسىدا ھېچقانداق خەت كۆرۈنمەيدىكەن. ئىشلىتىش تەڭشەشكە بەك قۇلايسىزلىق ئېلىپ كېلىدىكەن. بۇنى ھەل قىلىش ئۇسۇلى vim قاچىلىغان مۇندەرىجىگە كىرىپ، menu_zh_cn.utf-8 نى Gvim دىئالوگ رامكىسىغا قايتا ئۇلاپ قويساقلا بولىدۇ.

www.uyghur-archive.com/ugubuntuputer:~$cd /usr/share/vim/vim72/lang
www.uyghur-archive.com/ugubuntuputer:~$sudo ln -s menu_zh_cn.utf-8.vim menu_zh_cn.utf8.vim

مەن يۇقىرىدا تونۇشتۇرۇپ ئۆتكەنلىرىم، vim ئىشلىتىشتىكى ئەڭ ئاساسىي مەشغۇلات. vim نى تۇللۇق تونۇشتۇرۇش ئۈچۈن يازسا بىر كىتاب چىقىدۇ. قالغان ئىقتىدارلىرى ئۈستىدە ۋاقتى كەلگەندە ئايرىم توختىلىمەن.
vi/vim سەپلىمە ھۆججىتىنىڭ ئورنى vimrc./~ بۇلۇپ (بولمىسا قورۇڭلار)، تۆۋەندىكىسى مەن بىر نەچچە يىل داۋامىدا ئۇچىراتقان ئەڭ ياخشى سەپلىمە تەڭشىكى ئىكەن، ئىشلىىپ باقارسىلەر. ھەقىقەتەن بەك ئىسىل.
